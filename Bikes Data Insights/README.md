**🚴 Bike Store Sales Analysis using SQL**



**📌 Project Overview**

This project analyzes a Bike Store sales dataset using SQL to solve real business problems. The analysis focuses on customer behavior, product performance, sales trends, revenue analysis, and business reporting. The project demonstrates SQL concepts such as Joins, CTEs, Aggregate Functions, Window Functions, CASE Statements, Date Functions, and Business Analytics.





**Tools Used**

MySQL Workbench

Bikes Sales Dataset



**How to use this ?**



**Step 1**

Download this Folder.



**Step 2**

Open MySQL Workbench and create a new database.



CREATE DATABASE bikes;

USE bikes;



**Step 3**

Import all CSV files from the Dataset folder into their respective tables using Table Data Import Wizard in MySQL Workbench.



**Step 4**

Open the 01.sql file.



**Step 5**

Run each query to explore business insights and analyze the dataset.





**📌 Note**



The 2018 dataset contains only partial-year records. Therefore, year-over-year comparisons should be interpreted carefully by comparing equivalent time periods across years.
